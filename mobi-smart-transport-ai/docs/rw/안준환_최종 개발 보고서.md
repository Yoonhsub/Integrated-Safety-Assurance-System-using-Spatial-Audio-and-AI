## 4. 구현 완료 기능

- `BeaconSignal` 모델: 공식 JSON 필드 `beaconId`, `rssi`, `estimatedDistanceMeters`, `signalLevel`, `lastDetectedAt` 유지
- `BeaconSignalLevel` enum 직렬화: `VERY_CLOSE`, `CLOSE`, `MEDIUM`, `FAR`, `LOST` 유지
- `BeaconScanner` interface: `scan`, `stop`, `isScanning`, `targetBeaconId` filtering skeleton 정리
- `BeaconDistanceEstimator`: RSSI 기반 거리 추정, 추정 불가 시 null, signal level 분류 기준 정리
- `RssiMovingAverageSmoother`: 최근 RSSI window 기반 단순 이동 평균 helper 정리
- `DirectionReading` 모델: `headingDegrees`, `accuracy`, `updatedAt` JSON 계약 유지
- `DirectionAccuracy` enum 직렬화: `HIGH`, `MEDIUM`, `LOW`, `UNKNOWN` 유지
- `DirectionSensor` interface: readings stream, stop, isListening lifecycle skeleton 정리
- `DirectionReading.unknown()`: 센서값 부재 또는 정확도 판단 불가 fallback 정리
- public API export: `beacon_signal.dart`, `beacon_distance_estimator.dart`, `beacon_scanner.dart`, `direction_sensor.dart` export 유지
