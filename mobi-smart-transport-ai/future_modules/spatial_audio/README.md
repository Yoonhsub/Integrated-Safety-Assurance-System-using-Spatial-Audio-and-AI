# Spatial Audio - Future Module

공간음향 안내 기능의 향후 확장 프레임입니다.

4월에는 실제 HRTF/3D 렌더링을 구현하지 않습니다.
이 디렉터리는 향후 공간음향 기능을 위한 placeholder/계약 프레임만 보존합니다.
단, 요구사항명세서상 최종 시스템에 포함될 수 있으므로 위치와 계약 프레임을 보존합니다.

## 향후 예상 구조

```txt
Beacon/Bus Door Position
+ User Direction / Head Tracking
→ Spatial Audio Renderer
→ Directional Voice Cue
```
